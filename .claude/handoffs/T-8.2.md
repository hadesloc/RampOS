## Implementation Details

### 1. YieldProtocol Trait
```rust
#[async_trait]
pub trait YieldProtocol: Send + Sync {
    fn name(&self) -> &str;
    fn protocol_id(&self) -> ProtocolId;
    async fn current_apy(&self, token: Address) -> Result<f64>;
    async fn deposit(&self, token: Address, amount: U256) -> Result<H256>;
    async fn withdraw(&self, token: Address, amount: U256) -> Result<H256>;
    async fn balance(&self, token: Address) -> Result<U256>;
    async fn accrued_yield(&self, token: Address) -> Result<U256>;
    async fn claim_rewards(&self) -> Result<Option<H256>>;
    fn supports_token(&self, token: Address) -> bool;
    fn receipt_token(&self, token: Address) -> Option<Address>;
    async fn health_factor(&self) -> Result<f64>;
}
```

### 2. Aave V3 Integration
- Supply stablecoins (USDC, USDT, DAI) to Aave V3 Pool
- Track aTokens (receipt tokens)
- Claim AAVE incentive rewards
- Multi-chain support: Ethereum, Polygon, Arbitrum

### 3. Compound V3 Integration
- Supply to Compound V3 Comet markets
- USDC primary market support
- COMP reward claiming
- Multi-chain: Ethereum, Polygon, Arbitrum

### 4. YieldService Features
- **Auto-select best APY**: `deposit_to_best_protocol()`
- **Rebalancing**: Move funds to higher APY protocols
- **Health monitoring**: Emergency withdrawal if health factor drops
- **Yield reports**: Period-based yield analytics
- **Reward claiming**: `claim_all_rewards()` across protocols

### 5. Safety Controls
- Max allocation per protocol limits
- Min health factor monitoring (default 1.5)
- Emergency withdrawal capability
- Protocol-specific allocation tracking

## Usage Example

```rust
use ramp_core::r#yield::{
    YieldService, ProtocolRegistry, YieldAllocationConfig,
    AaveV3Protocol, AaveV3Addresses, CompoundV3Protocol, CompoundV3Addresses,
};

// Setup protocols
let mut registry = ProtocolRegistry::new();
let aave = AaveV3Protocol::new(1, AaveV3Addresses::ethereum_mainnet()?, account);
let compound = CompoundV3Protocol::new(1, CompoundV3Addresses::ethereum_mainnet()?, account);
registry.register(Box::new(aave));
registry.register(Box::new(compound));

// Create yield service
let config = YieldAllocationConfig::default();
let service = YieldService::new(registry, config);

// Deposit to best protocol
let (protocol, tx) = service.deposit_to_best_protocol(usdc_address, amount).await?;

// Rebalance when APY changes
let txs = service.rebalance(usdc_address).await?;

// Generate yield report
let report = service.generate_report(start_date, end_date).await?;
```
