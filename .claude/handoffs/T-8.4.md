# T-8.4: Stablecoin Price Oracle - Handoff

## Status: COMPLETED

## Summary
Implemented price oracle module with Chainlink integration and CoinGecko fallback for stablecoin valuation, including depeg detection and emergency pause capabilities.

## Files Created

### 1. `crates/ramp-core/src/oracle/mod.rs`
Main oracle module with:
- `Price` struct - Price data with token, chain, USD value, timestamp, source
- `PriceSource` enum - Chainlink, CoinGecko, Manual, Cached
- `DepegLevel` enum - Normal, Warning, Alert, Critical
- `DepegAlert` struct - Alert event with deviation info
- `PriceOracle` trait:
  ```rust
  #[async_trait]
  pub trait PriceOracle: Send + Sync {
      async fn get_price(&self, token: Address, chain_id: u64) -> Result<Price>;
      async fn get_prices(&self, tokens: Vec<Address>, chain_id: u64) -> Result<Vec<Price>>;
      fn is_stale(&self, price: &Price) -> bool;
      fn name(&self) -> &str;
  }
  ```
- `DepegConfig` - Configurable thresholds (warning 0.5%, alert 1%, critical 5%)
- `OracleRegistry` - Main registry with caching, fallback support, depeg detection

### 2. `crates/ramp-core/src/oracle/chainlink.rs`
Chainlink integration:
- `ChainlinkFeedConfig` - Feed addresses per chain/token
- `ChainlinkOracle` - Implementation with:
  - Pre-configured feeds for Ethereum, Polygon, Arbitrum
  - USDT, USDC, DAI price feeds
  - RPC endpoint configuration
  - Staleness checking (1 hour max)

### 3. `crates/ramp-core/src/oracle/fallback.rs`
CoinGecko fallback:
- `CoinGeckoConfig` - API configuration
- `CoinGeckoFallback` - Implementation with:
  - Known stablecoin ID mappings
  - Batch price fetching
  - 5-minute cache TTL
  - Token registration by symbol

### 4. Updated `crates/ramp-core/src/lib.rs`
- Added `pub mod oracle`
- Exported oracle types

## Key Features

### Depeg Detection
```rust
// Check deviation from $1.00 peg
let alert = registry.check_depeg(token, chain_id).await?;
if let Some(alert) = alert {
    match alert.level {
        DepegLevel::Warning => log_warning(alert),
        DepegLevel::Alert => send_notification(alert),
        DepegLevel::Critical => emergency_pause(alert),
    }
}
```

### Caching
- Configurable TTL (default 60 seconds)
- Automatic cache invalidation
- Returns cached prices with `PriceSource::Cached` marker

### Fallback Chain
```rust
let registry = OracleRegistry::new(Arc::new(ChainlinkOracle::with_default_config()))
    .with_fallback(Arc::new(CoinGeckoFallback::with_default_config()))
    .with_cache_ttl(Duration::from_secs(30));
```

### Emergency Pause
```rust
if registry.should_emergency_pause(token, chain_id).await? {
    // Deviation > 5%, pause operations
    pause_stablecoin_operations(token);
}
```
