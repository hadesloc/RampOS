## Default Tier Limits (SBV Compliant)

| Tier | Single TX | Daily | Monthly | Description |
|------|-----------|-------|---------|-------------|
| Tier 0 | 0 | 0 | 0 | Unverified - no transactions |
| Tier 1 | 50M VND | 100M VND | 1B VND | Basic eKYC verified |
| Tier 2 | 200M VND | 500M VND | 5B VND | Enhanced KYC verified |
| Tier 3 | 1B VND | Unlimited | Unlimited | Premium/Business |

## Key Features

1. **Vietnam Timezone Reset**
   - Daily limits reset at midnight Vietnam time (Asia/Ho_Chi_Minh)
   - Monthly limits reset on 1st of each month Vietnam time

2. **Configurable Enforcement**
   - Can enable/disable enforcement on payin/payout separately
   - Custom tier limits per tenant
   - User-level limit overrides with approval tracking

3. **Multi-tenant Isolation**
   - RLS policies on all new tables
   - Tenant-scoped configuration

4. **Audit Trail**
   - Custom limit changes tracked with reason and approver
   - Transaction history maintained for compliance

## Integration Points

### To use in transaction flow:
```rust
use ramp_compliance::limits::{VndLimitChecker, VndLimitConfig};

// Check before transaction
let result = limit_checker.check_limit(
    &tenant_id,
    &user_id,
    amount_vnd,
    "PAYIN_VND"
).await?;

if !result.is_approved() {
    // Handle limit exceeded
}

// Record after successful transaction
limit_checker.record_transaction(
    &tenant_id,
    &user_id,
    amount_vnd,
    "PAYIN_VND"
).await?;
```
