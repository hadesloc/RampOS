## Implementation Details

### 1. YieldStrategy Trait
```rust
#[async_trait]
pub trait YieldStrategy: Send + Sync {
    fn config(&self) -> &StrategyConfig;
    fn id(&self) -> StrategyId;
    async fn should_rebalance(...) -> Result<bool>;
    async fn calculate_allocation(...) -> Result<Vec<(ProtocolId, U256)>>;
    async fn execute_rebalance(...) -> Result<Vec<H256>>;
    async fn check_emergency_exit(...) -> Result<Vec<ProtocolId>>;
    fn estimate_rebalance_gas(&self, num_operations: usize) -> U256;
}
```

### 2. Strategy Configurations

| Strategy | Risk | Max Protocol | Min Health | Rebalance Threshold | Interval |
|----------|------|--------------|------------|---------------------|----------|
| Conservative | Low | 40% | 2.0 | 1.0% | Daily |
| Balanced | Medium | 50% | 1.5 | 0.5% | Hourly |
| Aggressive | High | 70% | 1.2 | 0.3% | 30 min |

### 3. Auto-Rebalancing Features
- Monitor APY changes at configured intervals
- Rebalance when APY difference exceeds threshold
- Gas-aware rebalancing (optional per strategy)
- Minimum rebalance amount to avoid dust transactions
- Track last rebalance timestamp per token

### 4. Risk Controls
- Protocol exposure limits (max % per protocol)
- Token concentration limits
- Minimum health factor monitoring
- Emergency exit triggers when health drops

### 5. API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/v1/yield/strategies` | List all available strategies |
| GET | `/v1/yield/strategies/:id` | Get strategy details |
| POST | `/v1/yield/strategies/:id/activate` | Activate a strategy |
| POST | `/v1/yield/strategies/:id/deactivate` | Deactivate a strategy |
| GET | `/v1/yield/performance` | Get yield performance metrics |
| POST | `/v1/yield/rebalance` | Trigger manual rebalance |
| GET | `/v1/yield/apys` | Get current APYs across protocols |

### 6. StrategyManager
- Manages strategy lifecycle (activate/deactivate)
- Tracks rebalance history
- Coordinates auto-rebalancing across tokens
- Handles emergency exits

## Usage Example

```rust
use ramp_core::r#yield::{
    StrategyManager, StrategyId, YieldService, ProtocolRegistry,
};

// Create strategy manager
let manager = StrategyManager::new();

// List available strategies
let strategies = manager.list_strategies();

// Activate balanced strategy
manager.activate_strategy(StrategyId::Balanced)?;

// Auto-rebalance check
let txs = manager.auto_rebalance(
    &yield_service,
    &registry,
    usdc_address,
    &current_allocations,
).await?;

// Check emergency exits
let emergency_txs = manager.check_emergency_exits(
    &yield_service,
    &registry,
    &[usdc_address],
).await?;
```
