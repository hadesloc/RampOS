## Deployment Commands

### Testnet (Base Sepolia)

```bash
# Set environment variables
export DEPLOYER_PRIVATE_KEY=0x...
export PAYMASTER_SIGNER=0x...
export BASE_SEPOLIA_RPC_URL=https://sepolia.base.org
export BASESCAN_API_KEY=...

# Deploy
forge script script/DeployBase.s.sol:DeployBaseTestnet \
  --rpc-url base_sepolia \
  --broadcast \
  --verify

# Verify deployment
export FACTORY_ADDRESS=0x...
export PAYMASTER_ADDRESS=0x...
forge script script/VerifyBase.s.sol:VerifyBaseDeployment \
  --rpc-url base_sepolia
```

### Mainnet

```bash
# Set environment variables
export DEPLOYER_PRIVATE_KEY=0x...
export PAYMASTER_SIGNER=0x...
export BASE_RPC_URL=https://mainnet.base.org
export BASESCAN_API_KEY=...
export PAYMASTER_DEPOSIT=100000000000000000  # 0.1 ETH

forge script script/DeployBase.s.sol:DeployBaseMainnet \
  --rpc-url base \
  --broadcast \
  --verify

forge script script/VerifyBase.s.sol:VerifyBaseDeployment \
  --rpc-url base
```

## Verification Steps

1. **Factory Verification**
   - Entry point address matches v0.7
   - Account implementation deployed
   - Counterfactual address computation works

2. **Paymaster Verification**
   - Entry point configured correctly
   - Owner and signer set
   - Deposit funded

3. **Entry Point Verification**
   - Contract deployed on Base
   - Code size > 0

## Gas Considerations

Base has very low gas fees:
- Typical `maxFeePerGas`: 0.1 gwei
- Typical `maxPriorityFeePerGas`: 0.001 gwei
- Most transactions cost < $0.01
