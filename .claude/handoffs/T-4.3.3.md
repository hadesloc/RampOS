## Configuration
- Default Base URL: `http://localhost:3000/v1`
- Default API Key: `test_key`
- Defined thresholds:
  - p95 < 300ms (GET)
  - p95 < 500ms (POST)
  - Error rate < 1%

## Usage
Run tests using k6:
```bash
k6 run tests/load/payin.js
k6 run -e SCENARIO=load tests/load/payout.js
```
