## Key Features
- **CaseNote Struct**: Stores note data (id, case_id, author, content, type, internal flag).
- **NoteType Enum**: Supports Comment, StatusChange, EvidenceAdded, Decision, Escalation.
- **CaseNoteManager**:
    - `add_note`: Adds a note manually.
    - `on_status_change`: Auto-adds note when case status changes.
    - `on_assignment_change`: Auto-adds note when assignment changes.
    - `on_resolution`: Auto-adds note when case is resolved.
