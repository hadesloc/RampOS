### Bundler Support
- **Pimlico**: Primary bundler for both mainnet and testnet
- **Stackup**: Alternative bundler for mainnet

### Gas Estimation
- Base gas multiplied by 1.2x for zk-proof overhead
- PreVerificationGas uses 1.5x multiplier due to higher zkEVM costs

## Deployment Commands

### Mainnet
```bash
forge script script/DeployPolygonZkEVM.s.sol:DeployPolygonZkEVMMainnet \
  --rpc-url $POLYGON_ZKEVM_RPC_URL --broadcast --verify \
  --verifier blockscout --verifier-url https://zkevm.polygonscan.com/api
```

### Testnet (Cardona)
```bash
forge script script/DeployPolygonZkEVM.s.sol:DeployPolygonZkEVMTestnet \
  --rpc-url $POLYGON_ZKEVM_CARDONA_RPC_URL --broadcast --verify \
  --verifier blockscout --verifier-url https://cardona-zkevm.polygonscan.com/api
```

## Post-Deployment Steps

1. Fund Paymaster with ETH for gas sponsorship
2. Add backend address as VND Token minter
3. Configure bundler with Pimlico or Stackup API key
4. Update .env with deployed contract addresses
5. Test userOp submission through bundler

## Testing Notes

- Polygon zkEVM uses ETH as native token (bridged from Ethereum)
- EIP-4337 EntryPoint v0.6 is at standard address
- Transaction finality is faster than Ethereum L1
- Gas estimation differs due to zk-proof overhead
