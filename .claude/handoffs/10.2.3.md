# Task 10.2.3: Add accurate gas estimation

## Implemented Features
- `GasEstimator` struct for estimating UserOperation gas limits.
- `GasEstimate` struct to hold results.
- `estimate_pre_verification_gas`: Calculates overhead based on calldata size and fixed costs (21000 + packed overhead).
- `estimate_call_gas`: Simulates the call using `eth_estimateGas`.
- `estimate_verification_gas`: Currently uses a safe default (100,000) as full simulation requires EntryPoint ABI and revert handling.
- `estimate_fees`: Fetches EIP-1559 fees from the network.
