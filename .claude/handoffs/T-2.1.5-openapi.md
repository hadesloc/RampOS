# OpenAPI Implementation

## Overview
Implemented OpenAPI documentation using `utoipa` and served it via Swagger UI.

## Changes
1. Added dependencies: `utoipa` and `utoipa-swagger-ui`
2. Created `openapi.rs` with API documentation spec
3. Added `ToSchema` derives to all DTOs in `dto.rs`
4. Annotated handlers with `#[utoipa::path]` in `handlers/`
5. Registered OpenAPI routes in `router.rs`
   - `/swagger-ui` - Interactive documentation
   - `/api-docs/openapi.json` - Raw OpenAPI spec

## Endpoints Documented
- `POST /v1/intents/payin`
- `POST /v1/intents/payin/confirm`
- `POST /v1/intents/payout`
- `GET /v1/intents/:id`
- `GET /v1/intents`
- `POST /v1/events/trade-executed`
- `GET /v1/users/:tenant_id/:user_id/balances`
- `GET /health`
- `GET /ready`
