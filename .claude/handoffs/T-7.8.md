## License Types Supported
1. **EXCHANGE** - Crypto exchange license (5 requirements, 4 mandatory)
2. **CUSTODIAL** - Digital asset custody license (5 requirements, all mandatory)
3. **PAYMENT** - Payment service provider license (4 requirements, all mandatory)

## Status Workflow
```
DRAFT -> SUBMITTED -> UNDER_REVIEW -> APPROVED -> ACTIVE
                                   -> REJECTED -> DRAFT (resubmit)
                          ACTIVE -> EXPIRED (auto)
                                 -> SUSPENDED
                                 -> REVOKED
```
