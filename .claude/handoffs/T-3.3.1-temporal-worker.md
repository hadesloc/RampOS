# Task Handoff: T-3.3.1 Temporal Worker Setup

## Summary
Implemented the Temporal worker infrastructure for RampOS, including:
1. Created `TemporalWorker` service that manages workflow execution
2. Implemented workflow definitions for Payin, Payout, and Trade
3. Added simulated activity execution with integration to repositories
4. Implemented signal handling for bank confirmations and settlement results
5. Created `WorkflowClient` for easy interaction from other services

## Key Components

### 1. Workflow Definitions (`crates/ramp-core/src/workflows/mod.rs`)
- `PayinWorkflowInput` / `PayinWorkflowResult`
- `PayoutWorkflowInput` / `PayoutWorkflowResult`
- `TradeWorkflowInput` / `TradeWorkflowResult`
- `payin_activities`, `payout_activities`, `trade_activities` modules

### 2. Temporal Worker (`crates/ramp-core/src/workflows/worker.rs`)
- `TemporalWorker`: Main worker service that polls for tasks
- `TemporalWorkerConfig`: Configuration from environment variables
- `WorkflowClient`: Client for starting workflows and sending signals

### 3. Simulation Mode
The current implementation runs in "simulation mode" using in-process task queues and channels. This allows for testing and development without a running Temporal server. For production, the worker should be updated to use the actual `temporal-sdk` crate.

## Configuration
Environment variables:
- `TEMPORAL_SERVER_URL`: URL of the Temporal server (default: http://localhost:7233)
- `TEMPORAL_NAMESPACE`: Temporal namespace (default: rampos)
- `TEMPORAL_TASK_QUEUE`: Task queue name (default: rampos-workflows)

## Usage Example

```rust
use ramp_core::workflows::worker::{TemporalWorker, TemporalWorkerConfig, WorkflowClient};

// Initialize worker
let config = TemporalWorkerConfig::from_env();
let worker = Arc::new(TemporalWorker::new(config, intent_repo, ledger_repo, event_pub));

// Start worker in background
tokio::spawn(async move {
    worker.run().await.unwrap();
});

// Start workflow
let client = WorkflowClient::new(worker.clone());
let workflow_id = client.start_payin(input).await?;
```
