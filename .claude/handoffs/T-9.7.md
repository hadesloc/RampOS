## Usage Examples

### Deploy to Optimism Sepolia (Testnet)
```bash
forge script script/DeployOptimism.s.sol:DeployOptimism \
  --rpc-url https://sepolia.optimism.io \
  --broadcast --verify
```

### Deploy to Optimism Mainnet (Production)
```bash
forge script script/DeployOptimism.s.sol:DeployOptimism \
  --rpc-url https://mainnet.optimism.io \
  --broadcast --verify
```

### Fund Paymaster
```bash
PAYMASTER_ADDRESS=0x... FUND_AMOUNT=100000000000000000 \
forge script script/DeployOptimism.s.sol:FundOptimismPaymaster \
  --rpc-url https://sepolia.optimism.io \
  --broadcast
```

### Verify Contracts
```bash
forge verify-contract <FACTORY_ADDRESS> RampOSAccountFactory \
  --chain optimism \
  --constructor-args $(cast abi-encode "constructor(address)" 0x5FF137D4b0FDCD49DcA30c7CF57E578a026d2789)
```

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `DEPLOYER_PRIVATE_KEY` | Yes | Private key for deployment |
| `OPTIMISM_RPC_URL` | Yes | RPC endpoint (mainnet or testnet) |
| `PAYMASTER_SIGNER` | No | Signer for paymaster (defaults to deployer) |
| `USE_ENTRYPOINT_V7` | No | Set to "true" to use v0.7 EntryPoint |
| `PAYMASTER_ADDRESS` | For funding | Deployed paymaster address |
| `FUND_AMOUNT` | For funding | Amount in wei (default: 0.1 ETH) |

## Post-Deployment Checklist

1. **Fund Paymaster with ETH**
   ```bash
   cast send <PAYMASTER_ADDRESS> --value 0.1ether
   ```

2. **Deposit to EntryPoint**
   ```solidity
   paymaster.deposit{value: 0.1 ether}();
   ```

3. **Add backend as VND Token minter**
   ```solidity
   vndToken.grantRole(MINTER_ROLE, backendAddress);
   ```

4. **Configure bundler endpoint** in backend environment

5. **Update environment variables** with deployed addresses
