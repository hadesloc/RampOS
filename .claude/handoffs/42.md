# Task 42 Handoff: Portal Sidebar Test Failure Fix

## Scope
Fix failing tests in portal sidebar module only.

## Root Cause
There were two independent issues causing the failing test state:

1. **Undefined component at runtime**
   - `PortalSidebar` imported `LocaleSwitcher` as a **named export**:
     - `import { LocaleSwitcher } from "@/components/locale-switcher";`
   - But `locale-switcher.tsx` exports a **default** function.
   - This made `LocaleSwitcher` evaluate to `undefined`, triggering:
     - `Element type is invalid ... got: undefined`.

2. **Route mocking mismatch in tests**
   - Component uses `usePathname` from `@/navigation` (next-intl wrapper).
   - Test mocked `usePathname` from `next/navigation`.
   - Active-route assertions failed because the mocked hook did not affect the hook actually used by the component.
