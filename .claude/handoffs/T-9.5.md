## Implementation Details

### DeployArbitrum.s.sol Structure

The script contains 5 deployment contracts:

1. **DeployArbitrum** (Main)
   - Deploys VNDToken, RampOSAccountFactory, RampOSPaymaster
   - Supports both EntryPoint v0.6 and v0.7
   - Chain validation for Arbitrum One (42161) and Sepolia (421614)
   - Comprehensive post-deployment checklist

2. **DeployArbitrumFactory**
   - Deploys only the Account Factory
   - For partial deployments or upgrades

3. **DeployArbitrumPaymaster**
   - Deploys only the Paymaster
   - Configurable signer address

4. **VerifyArbitrum**
   - Outputs verification commands for Arbiscan
   - Supports both mainnet and testnet

5. **FundPaymaster**
   - Funds paymaster with ETH
   - Deposits to EntryPoint for gas sponsorship

## Usage

### Deploy to Arbitrum Sepolia (Testnet)

```bash
forge script script/DeployArbitrum.s.sol:DeployArbitrum \
  --rpc-url https://sepolia-rollup.arbitrum.io/rpc \
  --broadcast --verify
```

### Deploy to Arbitrum One (Mainnet)

```bash
forge script script/DeployArbitrum.s.sol:DeployArbitrum \
  --rpc-url https://arb1.arbitrum.io/rpc \
  --broadcast --verify
```

### Environment Variables Required

```bash
# Required
DEPLOYER_PRIVATE_KEY=0x...

# Optional
USE_ENTRYPOINT_V7=true          # Default: false (uses v0.6)
PAYMASTER_SIGNER=0x...          # Default: deployer address
ARBISCAN_API_KEY=...            # For contract verification
```

### Fund Paymaster

```bash
PAYMASTER_ADDRESS=0x... \
FUND_AMOUNT=100000000000000000 \  # 0.1 ETH in wei
forge script script/DeployArbitrum.s.sol:FundPaymaster \
  --rpc-url https://arb1.arbitrum.io/rpc \
  --broadcast
```
