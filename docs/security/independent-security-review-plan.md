# Independent Security Review Plan

## Review Scope
- Partner governance and config governance
- Corridor/canonical payment paths
- Compliance routing and institutional evidence
- Treasury/reconciliation/audit controls
- Liquidity scoring and explainability
- Certification and break-glass flows

## Required Outputs
- finding id
- severity
- owner
- due date
- closure evidence
- explicit risk acceptance if not fixed

## Signoff Rule
- No unresolved `high` or `critical` findings without named approver and expiry.
