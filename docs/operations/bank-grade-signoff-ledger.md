# Bank-Grade Signoff Ledger

## Required Evidence Categories
- release hardening
- compatibility proof
- staging validation
- migration/rollback rehearsal
- backup/restore and disaster recovery
- independent security review
- operational runbooks

## Required Fields
- candidate SHA
- scope
- approvers
- approval timestamps
- exceptions
- exception expiry
- export location

## Gate
- A candidate is not bank-grade until every required category is attached and approved.
