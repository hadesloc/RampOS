# Disaster Recovery Plan

## Scope
- Database recovery
- Export/audit recovery
- Secret/config restoration
- Critical admin surfaces required for operator control

## Rehearsals
- Backup restore rehearsal
- Rollback rehearsal
- Recovery time measurement
- Evidence capture and signoff

## Proof Required
- recovery start/end timestamps
- restored dataset identifier
- verification commands after restore
- operator attribution
- unresolved exceptions
